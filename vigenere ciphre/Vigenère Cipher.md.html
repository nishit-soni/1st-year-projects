Encryption Technique

Here‚Äôs a step-by-step description of the process to encrypt a plaintext:

  1. Choose a Keyword: Select a keyword or keyphrase. This keyword is repeated to match the length of the plaintext. For example, if the plaintext is longer than the keyword, the keyword is repeated from the beginning to match the length of the plaintext.

  2. Assign Numerical Values: Convert each letter of the keyword and the plaintext to their corresponding numerical values. This is often done using a standard numbering scheme, such as A=0, B=1, ‚Ä¶, Z=25.

  3. Encrypt Each Letter: For each letter in the plaintext, apply the following steps:

    • Determine the shift: Find the corresponding letter in the keyword.
    • Shift the plaintext letter: Add the numerical values of the plaintext letter and the keyword letter together (mod 26).
    • Convert back to a letter: Convert the resulting numerical value back to a letter using the same numbering scheme.

  4. Repeat: Continue encrypting each letter of the plaintext using the corresponding letters in the repeated keyword until the entire plaintext is encrypted.

Decryption Technique

The decryption is the exact same process as encryption , with a small change : instead of adding the numerical value of the keyword , we subtract it form plaintext‚Äôs alphabet.
